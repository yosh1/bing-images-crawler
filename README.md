# keras-face-recognition

---

## get images

### package install

```
$ pip3 install icrawler
```

### run

```
$ python3 ./get_images.py <ARG1: SAVE_FILE_DIR> <ARG2: SEARCH_WORD> <ARG3: SAVE_FILE_NUM>
```

## detect images

### package install

```
$ pip3 install opencv-python
```

### run

```
$ python3 ./detect_face.py
```
